## ✨ Features & Innovations
- **2-Tier Enterprise Navigation**: Executive hub with dynamic weather badge and 5 dedicated operational tabs (`Interactive Map`, `Policy Simulator`, `Source Attribution`, `Enforcement Directives`, `Multi-City Benchmarking`).
- **Geospatial Air Quality Map across 32 Gujarat Nodes**: Leaflet dark matter map featuring bounding heatmaps, sensor pins, and clickable nodes driving right-panel analytics.
- **72-Hour Comparative Dispersion Curves**: Dynamic coordinate-mapped SVG kinetic trajectories comparing `Baseline` vs. `Post-Intervention` curves under 4 active municipal policy toggles.
- **24/7 Multilingual AI Support Chatbot**: Powered by `Groq Llama-3.1-8b-instant`. Includes explicit language selector (`English`, `Gujarati/ગુજરાતી`, `Hindi/हिंदी`) for citizen guidance.

## 🛠️ Technology Stack
- **Frontend**: Next.js 16 (Turbopack, App Router, TypeScript, React 19), Tailwind CSS, Framer Motion, Leaflet & React-Leaflet (`CartoDB Dark Matter`), Lucide React icons.
- **Backend Communication**: Sub-second parallel `asyncio.gather` endpoints (`FastAPI + Uvicorn + Groq Multi-Agent Engine`).
