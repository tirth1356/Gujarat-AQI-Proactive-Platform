This is the Next.js 16 (App Router + Turbopack + TypeScript + Tailwind CSS + Leaflet) frontend for the **Gujarat Air Quality Intelligence Platform**.

## Getting Started Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
